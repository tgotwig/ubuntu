# ubuntu

```sh
docker run -it --rm tgotwig/ubuntu
```

Ubuntu 24.04 with the following extra features:

- asdf
- atuin
- bat / batcat
- build-essential (cc)
- curl
- dnsutils (delv, **dig**, host, mdig, **nslookup**, nsupdate)
- docker
- dust
- fd /fdfind
- fish
- git
- gnupg
- iproute2 (bridge, ctstat, **ip**, lnstat, nstat, rtmon, ss, tc)
- jq
- less
- lsd
- mediainfo
- micro
- nc
- nmap
- ping
- ripgrep / rg
- sd
- ssh
- starship
- task (go-task)
- tree
- unzip
- wget
- z
